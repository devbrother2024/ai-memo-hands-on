# Story 4.2: 노트 내용 기반 자동 요약 생성

## Status

Done

## Story

**As a** 노트 작성자,
**I want** 노트 내용을 바탕으로 AI가 자동으로 3-6개 불릿 포인트 요약을 생성하는 기능,
**so that** 긴 노트의 핵심 내용을 빠르게 파악하고 정보를 효율적으로 관리할 수 있다.

## Acceptance Criteria

1. 노트 내용이 입력/수정될 때 자동 요약 생성 버튼이 표시되어야 한다
2. 요약 생성 시 3-6개의 불릿 포인트로 구성된 요약이 생성되어야 한다
3. 요약 생성 중에는 로딩 상태가 표시되어야 한다
4. 생성된 요약은 노트와 함께 저장되어야 한다
5. 요약 생성 실패 시 적절한 에러 메시지가 표시되어야 한다
6. 요약 재생성 기능이 제공되어야 한다
7. 토큰 사용량이 8k를 초과하지 않도록 노트 내용이 제한되어야 한다
8. 요약 품질이 일관성 있게 유지되어야 한다
9. 요약 생성은 10초 이내에 완료되어야 한다
10. 생성된 요약은 사용자가 수동으로 편집할 수 있어야 한다
